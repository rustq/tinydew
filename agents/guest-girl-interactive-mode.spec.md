# Guest Interactive Mode Spec

## Status
Implemented.

## Behavior
- Interactive runtime enables guest control mode.
- Active control in interactive flow is guest-only.
- Arrow keys move guest; player remains independently tracked.
- Space triggers guest greeting message.
- On Spring Day 28, guest greeting is:
  `✨ Happy Butterfly Festival!`
- Guest transitions across Farm/EastPath/Square/SouthRiver follow map gates.
- Guest obeys non-walkable tiles, including Fountain and Wonder.
